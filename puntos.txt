esto es para gitbash

1 echo "lo que quieres adentro del archivo" > "tipo de archivo" --> para crea un archivo dentro de una carpeta con gitbash.
2. git init --> crear un repositorio en una carpeta.
3. code . --> abrir visual code.
4. git status --> estado del repositorio.
5. git commit -m "lo que dices cuando hiciste un cambio en algo que es lo que cambio" --> sirve para cuando hacer un cambio "añadir algo" todos los commit tienen que tener un mensaje.
6. git add . o git add "referirse al tipo de archivo" para añadir un cambio.
7. git log --> es como un historial.
8. git checkout --> para revisar cambios especificos que hiciste, copiando lo que esta al lado del commit cuando miras el log.
9. git branch --> visualización de ramas.
10. git switch master --> es como para devolverse de la rama en la que estabas metido.
11. git config --global user.name "el nombre que ud le quiere dar"
12. git config --global user.email "el correo que ud quiere dar"
